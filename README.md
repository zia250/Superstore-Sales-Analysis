# 🛒 Superstore Sales Performance Analysis

## 📌 Project Overview
This project analyzes sales, profit, and customer trends 
of a retail superstore using multiple data analysis tools.

## 🛠️ Tools Used
- **MySQL** — Data storage & querying
- **Python** — Data cleaning & EDA
- **Power BI** — Interactive dashboard
- **Excel** — Summary & pivot tables

## 📊 Key Insights
- Technology category generates highest sales
- West region is most profitable
- Heavy discounts reduce profit significantly
- Q4 (October-December) shows highest sales every year

## 📁 Files
| File | Description |
|---|---|
| `Sample - Superstore.csv` | Raw dataset |
| `queries.sql` | MySQL analysis queries |
| `Sale_Analysis.ipynb` | Python EDA notebook |
| `Sale_Analysis_Dashboard.pbix` | Power BI dashboard |
